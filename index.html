主菜

豚こまとキャベツのうま塩炒め

豚バラがなくても豚こまでOK。にんにく塩だれで間違いないやつ。

使う食材：豚こま・キャベツ・にんにく

つくりかたを見る

材料（2人分）

  • 豚こま肉 … 200g
  • キャベツ … 1/4玉
  • にんにく … 1かけ
  • ごま油 … 大さじ1
  • 鶏ガラスープの素 … 小さじ1
  • 塩こしょう … 少々

手順

  1. キャベツはざく切り、にんにくはみじん切りにする
  2. フライパンにごま油とにんにくを熱し、豚こまを炒める
  3. 肉の色が変わったらキャベツを加えて強火でさっと炒める
  4. 鶏ガラスープの素と塩こしょうで味をととのえて完成

副菜1

にんじんとツナの無限サラダ

レンジ2分。あえるだけで一品増える優秀なやつ。

使う食材：にんじん・ツナ缶

つくりかたを見る

材料（2人分）

  • にんじん … 1本
  • ツナ缶 … 1缶
  • ごま油 … 小さじ2
  • 鶏ガラスープの素 … 小さじ1/2
  • 白ごま … 適量

手順

  1. にんじんを細切りにして耐熱ボウルに入れる
  2. ふんわりラップをして電子レンジ（600W）で2分加熱
  3. 油を切ったツナと調味料を混ぜ、白ごまをふって完成

副菜2

やみつき塩だれきゅうり

たたいて和えるだけ。火を使わないからゼロ労力。

使う食材：きゅうり・ごま油・塩昆布

つくりかたを見る

材料（2人分）

  • きゅうり … 2本
  • 塩昆布 … ひとつまみ
  • ごま油 … 小さじ2
  • 塩 … 少々

手順

  1. きゅうりをめん棒などでたたき、食べやすい大きさに割る
  2. ポリ袋に材料をすべて入れて振り混ぜる
  3. 5分ほどなじませて完成

汁物

キャベツと卵のふわたまスープ

余ったキャベツはスープへ。卵でやさしい味に。

使う食材：キャベツ・卵・鶏ガラスープの素

つくりかたを見る

材料（2人分）

  • キャベツ … 2枚
  • 卵 … 1個
  • 水 … 400ml
  • 鶏ガラスープの素 … 大さじ1/2
  • 塩こしょう … 少々

手順

  1. 鍋に水と鶏ガラスープの素を入れて沸かす
  2. ざく切りにしたキャベツを加えて2分ほど煮る
  3. 溶き卵を回し入れ、ふんわり浮いてきたら塩こしょうでととのえる
